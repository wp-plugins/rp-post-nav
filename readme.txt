Go to Plugins > add new > search RP Post Nav > click Install2. Activate the plugin after installation3. Go to WordPress Dashboard > RP Post Nav== Frequently Asked Questions === Which is the good place to show Post Navigation? =It's either use automatic display or shortcode.== Screenshots ==1.
